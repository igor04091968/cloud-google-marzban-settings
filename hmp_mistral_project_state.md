# Состояние проекта HMP-Mistral-LLVM

## 1. Цель

Настроить локальную работу агента HMP (Hermetic Multi-Agent Platform), используя локально загруженную модель ИИ Mistral. Это позволит агенту работать автономно без обращения к внешним API.

## 2. Компоненты

*   **Агент:** HMP (Hermetic Multi-Agent Platform)
    *   **Расположение:** `/home/igor/gemini_projects/HMP/`
*   **Модель ИИ:** Mistral 7B Instruct v0.2 (GGUF)
    *   **Расположение:** `/home/igor/gemini_projects/models/mistral-7b-instruct-v0.2.Q4_K_M.gguf`
*   **Инференс-сервер (Планируется):** `llama.cpp` или аналогичный, который будет загружать GGUF-модель и предоставлять OpenAI-совместимый API эндпоинт. Компонент "LLVM" в названии, вероятно, относится к процессу компиляции и запуска инференс-сервера.

## 3. Текущий статус

*   **Модель ИИ:** Успешно загружена и сохранена локально.
*   **Агент HMP:** Проект клонирован, базовые настройки не изменялись для работы с локальной моделью.
*   **Инференс-сервер:** Не установлен и не настроен.

## 4. Следующие шаги

1.  **Установить и настроить `llama-cpp-python`:**
    *   Установить с поддержкой аппаратного ускорения, если возможно.
    *   Запустить его в режиме сервера, указав путь к файлу модели `mistral-7b-instruct-v0.2.Q4_K_M.gguf`.
    *   Убедиться, что сервер доступен по локальному адресу (например, `http://localhost:8000`).
2.  **Настроить агент HMP:**
    *   Найти конфигурационный файл агента HMP.
    *   Изменить эндпоинт API, на который он отправляет запросы, на адрес локального сервера `llama.cpp`.
    *   Изменить (если требуется) модель, используемую в запросах.
3.  **Запустить и протестировать:**
    *   Запустить агент HMP.
    *   Отправить тестовый запрос, чтобы убедиться, что он обрабатывается локальной моделью Mistral.
